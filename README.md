# Weather Diary

Автор: [Сербиев Артём]

Дата: [25.04.2026]

## Описание

Приложение "Weather Diary" позволяет вести дневник погоды: добавлять записи с датой, температурой, описанием и информацией об осадках, фильтровать их и сохранять данные в формате JSON.

## Функциональность

* Добавление записей о погоде.
* Просмотр всех записей в таблице.
* Фильтрация по дате.
* Сохранение и загрузка данных из JSON‑файла.
* Валидация ввода данных.

## Требования
- Python 3.6 или выше
- Стандартная библиотека Python (tkinter, json, os)
